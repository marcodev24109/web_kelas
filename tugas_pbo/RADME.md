# Website Kelas Info4 – Teknik Informatika

## Deskripsi Proyek
Proyek ini merupakan website sederhana untuk Kelas Info4 Angkatan 2024 Program Studi Teknik Informatika  
Website dibuat sebagai bagian dari tugas mata kuliah Pemrograman Berorientasi Objek (PBO)

Website menampilkan informasi kelas, akademik, serta media sosial dengan tampilan statis menggunakan HTML dan CSS  
Python digunakan sebagai web server sederhana untuk menjalankan website secara lokal

## Teknologi yang Digunakan
- HTML5 untuk struktur halaman
- CSS3 untuk tampilan dan layout
- Python untuk menjalankan server web lokal

## Cara Menjalankan Program
1. Pastikan Python sudah terinstall di komputer
2. Buka terminal atau command prompt
3. Masuk ke folder proyek
cd "tugas pbo"
4. Jalankan server Python
py app.py
5. Buka browser dan akses
http://localhost:8000



## Fungsi File
- `index.html` → Halaman utama website
- `sty.css` → Mengatur tampilan dan desain website
- `app.py` → Server web sederhana menggunakan Python
- `images/` → Menyimpan gambar pendukung website
- `icon/` → Menyimpan favicon website



## Tujuan Pembelajaran

- Memahami konsep client dan server
- Menggunakan Python sebagai web server sederhana
